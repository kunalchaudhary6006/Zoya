🎙️ Zoya – Cinematic AI Voice Assistant (Web-Based)

A futuristic, browser-based AI voice assistant with a cinematic UI, real-time voice interaction, and a witty Hinglish personality built for viral experiences.

🚀 Overview

Zoya is a fully client-side AI voice assistant that runs directly in your browser using modern Web APIs.
It combines stunning UI/UX, speech recognition, text-to-speech, and AI-powered conversations to create an engaging, entertaining, and highly interactive experience.

Designed with a viral-first approach, Zoya is perfect for:

🎬 Instagram Reels / YouTube Shorts
🤖 AI demos & experiments
🌐 Personal assistant prototypes
🧠 Voice UI research projects
